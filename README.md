# Infix-to-prefix-and-postfix-conversion
Here we are converting infix notation to a postfix or prefix notation.

Infix Expression	  Prefix Expression	  Postfix Expression
----------------    -----------------   -------------------
(A + B) * (C + D)	    * + A B + C D	    A B + C D + *
A * B + C * D	        + * A B * C D	    A B * C D * +
